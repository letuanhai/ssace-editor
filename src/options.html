Editor

Defaults for every Ace editor in SAS Studio (code editor, text viewer) and for the snippet editor below. Live-applies to open SAS Studio tabs on change. Theme is a dark/light pair (auto-switches with OS preference) - configured only here. Every other setting - including the keyboard handler (None/Vim/Emacs/Sublime/VSCode) - is configured from Ace's own stock settings panel (Ctrl-, in any editor, or the "Show settings menu" entry in the command palette when one is focused); changes made there now persist as the default for new editors and live-apply to open ones, same as this page.

Vim config

Applied to the shared vim keyboard handler (only takes effect once "Vim" is the active keyboard handler). One mapping per line, " starts a comment. Supported: map/nmap/imap/vmap <lhs> <rhs>, noremap/nnoremap/inoremap/vnoremap (same, non-recursive), and unmap/nunmap/iunmap/vunmap <lhs>. Anything else is skipped with a console warning. Removing a line here doesn't undo the mapping until the page is reloaded.

Snippets

Native Ace snippet format (snippet trigger / tab-indented body). Additive over the built-in SAS snippets. Live-applies to open SAS Studio tabs on Save.
